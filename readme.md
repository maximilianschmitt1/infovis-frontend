# infovis frontend

## Dependencies

* [node.js](https://nodejs.org/) or [io.js](https://iojs.org)

## Setup

```
$ git clone https://github.com/maximilianschmitt1/infovis-frontend
$ cd infovis-frontend
$ npm install
```

## Start

```
$ npm start
$ open http://localhost:3000
```

## Build

```
$ npm run build
```
